# 🌤️ Modern Weather Dashboard

A sleek, responsive, and high-performance Weather Application built using vanilla web technologies and integrating the OpenWeather API. Features a modern Glassmorphic user interface.

## 🚀 Features
- **Dynamic Search:** Query live weather conditions for any city globally.
- **Real-time Metrics:** Displays Temperature (°C), Humidity (%), Wind Speed (Km/h), and Description.
- **Smart Asset Loader:** Dynamically updates high-quality condition icons depending on current weather states.
- **Robust Error Architecture:** Gracefully catches 404 responses for missing cities and handles network failures.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Custom Glassmorphic Layout & Animations)
- JavaScript (ES6+ Asynchronous Fetch Architecture)
- OpenWeather Map API
